# GWT website - GitHub ready

Ovaj paket je pripremljen za GitHub Pages / Cloudflare Pages.

## Upload na GitHub
Uploadajte sadržaj ovog foldera u root repositoryja:
- index.html
- nova-cesta-65.html
- poslovni-prostori.html
- statusi.json
- assets/

## Promjena statusa stanova
Editirajte samo `statusi.json`.

Moguće vrijednosti:
- dostupno
- rezervirano
- prodano

Primjer:
```json
"S23": "prodano"
```

Nakon izmjene kliknite **Commit changes**.
